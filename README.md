# trade-doc-validation-system
An intelligent trade document validation system that automates cross-verification of critical logistics documents and flags inconsistencies using a weighted risk-based approach.
# Trade Document Validation System

## Overview
This project was developed during a hackathon to automate validation of international trade documents such as invoices, bills of lading, and customs declarations.

## Problem
Manual verification of trade documents is time-consuming and prone to errors, leading to fraud and compliance risks.

## Solution
A Python-based system that:
- Extracts key fields from documents
- Cross-validates multiple documents
- Assigns a risk score based on inconsistencies

## Features
- Multi-document validation (Invoice, BoL, CoO, Customs Declaration)
- Field comparison (HS code, value, quantity, consignee)
- Weighted risk scoring system
- HTML-based output interface
